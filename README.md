# Traffic Navigation Simulator

This project is a simple AI-based pathfinding application built with Streamlit that models a city as a graph, where nodes represent intersections and edges represent roads with costs. It allows users to find optimal routes between two points using Dijkstra (weighted shortest path) and BFS (unweighted shortest path), while also simulating traffic conditions through a “Rush Hour” mode that increases edge weights. The application provides an interactive interface and visualizes the graph with the computed path highlighted, demonstrating how graph theory and search algorithms can be applied to real-world navigation problems.
